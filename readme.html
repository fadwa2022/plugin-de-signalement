# Redmi pour le plugin Wordpress "Signalement" Le plugin "Signalement" permet d'afficher de manière dynamique la liste des menus du signal d'un post ou d'un article dans une page simple. Cette liste prend la forme d'un formulaire permettant aux utilisateurs de signaler un problème ou une erreur sur le site. # Fonctionnalités Ajouter une page de paramètres pour la partie administration du plugin dans le backoffice. Personnaliser l'icône du plugin. Faciliter la configuration des champs à conserver et des champs à supprimer du formulaire. Permettre la sauvegarde des données dans la base de données. Permettre l'utilisation d'un shortcode pour insérer le formulaire à l'endroit désiré. Activer et désactiver le plugin avec la création et la suppression de la table concernée dans la base de données. Enregistrer les signalements des utilisateurs dans la base de données. # Installation Télécharger le fichier ZIP du plugin depuis GitHub. Extraire le contenu du fichier ZIP dans le dossier "wp-content/plugins/" de votre site Wordpress. Activer le plugin dans le backoffice de Wordpress. # Utilisation Dans la page de paramètres du plugin, configurez les champs à conserver et à supprimer du formulaire selon vos préférences. Utilisez le shortcode [signalement-form] pour insérer le formulaire de signalement à l'endroit désiré sur votre site. Les signalements enregistrés par les utilisateurs seront disponibles dans la base de données et pourront être consultés dans le backoffice de Wordpress. Configuration
